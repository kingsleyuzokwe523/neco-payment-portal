# NECO Payment Portal

A payment request system where users submit payment requests and admin approves.

## Admin Login
- Username: `admin`
- Password: `admin123`

## Features
- User submits payment request with account details
- Admin approves requests
- User uploads payment receipt
- Admin confirms completion

## How to Use
1. Users submit payment request form
2. Admin approves from dashboard
3. User pays to NECO bank account
4. User uploads receipt
5. Admin marks as complete
